Conta e Configuração

Integrações

Beeno CRM — Central de Ajuda

Saiba como configurar webhooks e chaves API para aprimorar suas automações e integrações.

Quando usar API vs Webhook API (pull/consulta/ação): quando seu sistema precisa buscar dados na Beeno ou executar ações (criar/atualizar objetos, disparar automações, etc.).

Webhook (push/eventos): quando você quer ser notificado automaticamente sempre que algo acontecer (ex.: novo contato, mudança de fase, ticket resolvido, mensagem recebida, evento de automação executado).

Dica: combine ambos — use webhooks para reagir a eventos em tempo real e APIs para consultar detalhes adicionais quando necessário

Chaves de API

O que são Chaves de API são credenciais secretas que autenticam chamadas da sua aplicação aos endpoints da Beeno. Trate-as como senhas.

Como criar/excluir Acesse Configurações > Integrações > API Tokens. Clique em Criar Token. Defina o nome. Copie a chave (não será exibida novamente). Para excluir, localize a chave na lista e selecione Excluir. Boas práticas: use uma chave por aplicação/ambiente (prod, staging)

Boas práticas: Não exponha a chave em front-end, repositórios públicos ou prints. Rotacione periodicamente (ex.: a cada 90 dias) ou se houver suspeita de vazamento.

Limite de Requisições: 100 requisições a cada 10 segundos.

Quer iniciar a construção das suas integrações? Acesse nossa documentação em https://developers.beeno.com/

Webhooks

Como funcionam Webhooks enviam requisições HTTP POST para sua URL quando um evento ocorre na Beeno. Você configura quais eventos deseja assinar.

Eventos disponíveis Contato → criação, atualização ou exclusão. Negócio → criação, atualização ou exclusão. Empresa → criação, atualização ou exclusão. Tarefa → criação, atualização ou exclusão.

Como configurar no CRM Acesse Configurações > Integrações > Webhooks. Clique em Adicionar webhook. Informe Nome, URL do endpoint (HTTPS) e eventos Salve e teste.

Quer iniciar a construção das suas integrações? Acesse nossa documentação em https://developers.beeno.com/
